## O Party Time é uma aplicação desenvolvida com o objetivo de proporcionar uma experiência divertida e interativa para quem gosta de festas. Ele permite que os usuários organizem eventos, criem listas de convidados, gerenciem detalhes da festa, e mais. Este projeto foi criado para explorar conceitos de desenvolvimento front-end com React e back-end com Node.js.

### Funcionalidades
- Criação e gerenciamento de eventos.
- Adicionar e remover convidados das festas.
- Configurar detalhes como data, hora, local e tema da festa.

### Tecnologias Utilizadas
- React - Biblioteca JavaScript para construção de interfaces de usuário.
- Node.js - Ambiente de execução JavaScript server-side.
- Express - Framework web para Node.js.
- MySQL - Banco de dados relacional usado para armazenar as informações dos eventos e convidados.

### Instalação
Siga os passos abaixo para rodar o projeto localmente:

1. Clone o repositório:  
git clone https://github.com/Lafaietepedro/Party-Time.git

2. Navegue até o diretório do projeto:  
- cd Party-Time

3. Instale as dependências do front-end:  
- cd frontend  
- npm install

4. Instale as dependências do back-end:  
- cd backend  
- npm install

6. Configure as variáveis de ambiente no arquivo .env (por exemplo, as credenciais do banco de dados).

### Como Usar
1. Inicie o servidor do back-end:  
- cd backend  
- npm start
- 
2. Inicie o servidor do front-end:  
- cd frontend  
- npm start
 
3. Acesse o projeto no navegador via http://localhost:3000.
